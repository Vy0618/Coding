CSS - SELETORES (REGRAS)

COMO O CSS SABE ONDE APLICAR O ESTILO?

Seletor de TAG Seleciona todos os elementos de uma determinada tag HTML. p { color: blue; }

Seletor de CLASSE Seleciona elementos que têm um atributo class específico. .box { border: 1px solid black; }

Seletor de ID Seleciona o elemento com um id específico (único na página). #menu { background: gray; }

Seletor geral (*) Aplica as regras CSS a todos os elementos, independentemente do tipo, classe ou id. * { margin: 0; padding: 0; box-sizing: border-box; }
