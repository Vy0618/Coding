Como ligar o CSS ao HTML?

Meios de “linkar” o css com o html e qual é mais usado no mercado

Inline (Embutido): CSS é aplicado diretamente no elemento. <'p style="color: blue; font-size: 16px;">Este é um parágrafo com estilo inline.

Externo: CSS fica em um arquivo nome.CSS e você “linka” ele com esse arquivo através da TAG link. <'head> <'link rel="stylesheet" href="style.css">

Interna: CSS é embutido no head através da tag Style. <’!DOCTYPE html> <’html> <’head> <’style> ‘body { background’-color: ‘lightgray; } h1 ‘{ color: ‘navy; margin-left: ‘20px; } <’body> <’h1>Este é um título com estilo interno. <’p>Este é um parágrafo com estilo interno.
